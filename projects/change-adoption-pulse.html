Change Adoption Pulse

Enterprise rollout scenario for a new internal case/task management system across departments.

GitHub Repo Open Dataset CSV Back

This project tracks readiness, training progress, communication effectiveness, resistance signals, and milestone risk for Claims, Customer Service, Driver Licensing Support, IT Support, and Team Leads / Operations.

Scoring Core

Readiness Score (0-100)

1-5 scores are converted to 0-100 with (score - 1) / 4 * 100.

readiness = awareness*0.20 + understanding*0.20 + confidence*0.20 + benefit*0.15 + training*0.15 + manager_briefing*0.10

Adoption Risk Score (0-100, higher is worse)

Rule-based points:

+15 training < 70, +10 confidence < 3, +10 resistance > 3.5, +10 milestone delayed, +5 comms < 60, +5 open issues >= 4, +5 open risks >= 3, +5 perceived benefit < 3. Capped at 100.
