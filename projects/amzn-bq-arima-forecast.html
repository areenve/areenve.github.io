Problem

Build a reproducible forecasting pipeline where modeling runs in BigQuery, results are evaluated consistently, and visuals are publication-ready for the website case study.

Approach

  • Load and prepare AMZN time-series (frequency, missing values, train/test split).
  • Fit BigQuery ML ARIMA model (auto hyperparameters).
  • Generate forecasts + compare against holdout.
  • Plot forecast vs actual and summarize error metrics.
