AMZN BigQuery ARIMA Forecast

Time-series forecasting workflow with BigQuery ML ARIMA + evaluation-ready plots for the website case study.

GitHub Repo Back
Dataset
AMZN
Daily log returns pulled into BigQuery (train/eval split).
Model
ARIMA
BigQuery ML ARIMA (auto-ARIMA) trained in-warehouse.
Forecast horizon
7 days
Short-horizon forecast to validate end-to-end pipeline + reporting.

Problem

Build a reproducible forecasting pipeline where modeling runs in BigQuery, results are evaluated consistently, and visuals are publication-ready for the website case study.

Approach

  • Ingest AMZN price history into BigQuery and compute daily log returns.
  • Fit BigQuery ML ARIMA (auto-ARIMA) on the training window.
  • Generate a 7-day forecast and export results for the website report.
  • Produce a clean forecast visualization to communicate the pipeline output.

Results

AMZN daily log returns: recent history + 7-day forecast. Blue shows realized daily log returns; the vertical dotted line marks the forecast start. Orange is the ARIMA forecast for the next 7 trading days. This figure demonstrates the end-to-end BigQuery ML forecasting pipeline output in a website-ready format.

What this project demonstrates

  • BigQuery-first modeling workflow (no local model training required).
  • Clean separation between data prep, modeling SQL, and reporting outputs.
  • A single “hero” forecast figure for fast communication in a portfolio context.
